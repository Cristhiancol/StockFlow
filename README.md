## 🎯 Como Empezar

### 1. Clonar el Repositorio
```bash
git clone https://github.com/Cristhiancol/StockFlow.git
cd StockFlow
```

### 2. Configurar Entorno
```bash
python -m venv venv
source venv/bin/activate  # En Windows: venv\\Scripts\\activate
pip install -r requirements.txt
```
